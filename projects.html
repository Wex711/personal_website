Virtual Encylopedia Project

This project is an HTML, CSS, and JSON-ld based website which simulates the device known as a Pokédex from the Pokémon series. It includes information on over 100 pokemon, including appearance, cries, height, weight, types, and more.

Water Purity Tester

This project utilizes a MATLAB based app in combination with a microcontroller to control a machine that tests for water purity through the use of reagent test strips. It uses a vison system which is connected to the app through a cell phone camera to report water purity back to the app.
