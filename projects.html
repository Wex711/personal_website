Virtual Encyclopedia

This project is an HTML5, CSS, and JSON-LD based website which simulates the device known as a Pokédex from the Pokémon series. It includes information on over 100 Pokémon, including appearance, cries, height, weight, types, and more.

You can access it here.

Tech: HTML, CSS, JSON

Water Purity Tester

This project utilizes a MATLAB-based app in combination with a microcontroller to control a machine that tests for water purity through the use of reagent test strips. It uses a vision system connected to the app through a cell phone camera to judge water purity.

Tech: MATLAB, Arduino Board

Online Shoe Store

This project utilizes relationships between databases and draws upon SQL, PHP, and HTML5 via foreign and primary keys as well as CRUD functions to simulate the process of purchasing shoes from an online store where variables like stock, user information, and price are taken into account.

Tech: PHP, SQL, Filezilla, Localhost/XAMPP
